# Sudoku Generator & Solver
This project is a Python-based Sudoku application that can generate and solve Sudoku puzzles using an interactive graphical user interface (GUI).

## Features
- Generates random valid Sudoku puzzles every time
- Solves Sudoku puzzles using backtracking algorithm
- Interactive 9×9 grid for user input
- Fixed cells cannot be modified
- Real-time digit count display (1–9)
- Buttons for Solve, New Game, and Clear

## Technologies Used
- Python
- Tkinter (for GUI)
- Backtracking Algorithm
- Data Structures (Lists, Dictionaries)

## How to Run
1. Make sure Python is installed
2. Download the project files
3. Run the program:
   python sudoku_gui.py

## Project Description
The application generates a valid Sudoku board and removes some values to create a playable puzzle. Users can fill in the missing values, and the system ensures valid input. The solver uses a backtracking algorithm to find the correct solution.
